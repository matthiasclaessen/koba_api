## KOBA API

Dit is de opdracht voor de vacature van programmeur.

In het `api.php` bestand zitten de routes voor de API.

In het `ExpenseController.php` bestand zit de logica die dient om uitgaves op te halen, aan te maken en goed te keuren 
of af te keuren.

In het `Expense.php` bestand zit de logica voor de entiteit 'Expense'.

Tot slot zijn er ook verschillende migraties aangemaakt om de nodige tabellen aan te maken in de database.

Nadat de database is opgezet, kan je deze opvullen met dummydata door `php artisan db:seed` te draaien in de console.
